Description
My final project for this class will be a 2D top-down game focussed around zombie survival. The player will control a character in the middle of the screen and must survive using any means as long as they can. Zombies will continuously spawn from all directions and move towards the player as the player moves around the screen.
As stated earlier, the player can move around the screen and shoot bullets from a gun to eliminate the incoming zombies. If a zombie touches the player enough times, the game ends. The game will include a wave system to ensure that the game will eventually be hard for everyone that plays, increasing the difficulty by spawning more zombies or increasing their speed or health.
The goal of the game is to survive as many waves as possible. This will create a simple, but addictive gameplay that can theoretically go on forever as the player is constantly challenged to improve their performance against the zombies.
I have designed this project to start out simple for this class, but also have a lot of room to expand in the future. Future additions may include different zombie types, different weapons, and new game modes for the player to pick.

Technology
I will use the following technologies for this project:

Python (the programming language)
Visual Studio Code (IDE)
Pygame (for features of game)

I chose pygame for this project because it is well suited for the type of game i am trying to make, being 2d and top down. It allows easy movement across the screen, a straightforward collision detection, and decent rendering. I am using python because I am already familiar with the language, allowing me to focus on the game logic rather than learning a new language.
